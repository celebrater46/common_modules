FP COMMON MODULES について

アプリが増えてきたので、共通機能をまとめたライブラリを作りました。
いくつかのアプリは、このライブラリを一緒にアップロードしないと動きません。
詳細は作者のサイト（https://enin-world.sakura.ne.jp/）にてご確認ください。



##############################################################
使い方
##############################################################

ZIP をダウンロードしてお使いのサーバーにアップロードし、
各モジュールのフォルダ内にある init.sample.php の名前を init.php に変更して
お使いの環境に合わせて設定します。
